# IMDB-250-movies-budget
Script that scrapes imdb top 250 movies site asynchronously using python's asyncio library
## Project:
To scrape the list of imdb 250 movies list, to get the budget data of each movie.

## Libraries used
Ex. -

<b>Built with</b>
- asyncio and aiohttp
- pandas
